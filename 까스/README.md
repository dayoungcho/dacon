
사내 경영경제연구소의 연간보고서 자료(전력계통제약을 반영한 발전용 수요전망 모형 개발)에 따르면,
- 도시가스 수요, 발전용 수요 사이에 차이가 존재함.
- 도시가스 수요는 일정한 면이 있음(계절성이나 가격에 따른 변동 존재하긴 함) -> 예측이 용이하고, 실제 예측의 성능도 좋음
- 발전용 수요는 불규칙적임.(영향을 주는 요소는 신입 교육 자료 참고하자,,,)
- 실제로 시각화를 해보니 발전 공급량은 도시가스 공급량에 비해 규칙적이지 못한 양상이 보임.
- 그래서 그 보고서에서는 시계열 모형으로 수요를 예측한다.
- 저는 AI를 활용해보면 어떨까? 하는 아이디어에서, 
- 전력거래소의 일별 연료원별 전력거래량 데이터를 이용, TFT(temporal fusion transformer) 모델로 수요 예측 시도해보았다~~~
