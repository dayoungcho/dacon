
사내 경영경제연구소의 연간보고서 자료(전력계통제약을 반영한 발전용 수요전망 모형 개발)에 따르면,
1. 도시가스 수요, 발전용 수요 사이에 차이가 존재함.
2. 도시가스 수요는 일정한 면이 있음(계절성이나 가격에 따른 변동 존재하긴 함) -> 예측이 용이하고, 실제 예측의 성능도 좋음(가스공사에서 2021년에 도시가스 수요 예측 대회를 진행하기도 했음)
3. 그러나 발전용 수요는 불규칙적임. -> 영향을 주는 요소가 가변적이기 때문(신입 교육 자료 집어넣기)
4. 실제로 시각화를 해보니 발전 공급량은 도시가스 공급량에 비해 규칙적이지 못한 양상이 보임. (그래프 제시)
5. 그래서 저희가 참고한 연간보고서 자료에서는 시계열 모형으로 수요를 예측한다.
6. 저희 팀은 AI를 활용해보면 어떨까? 하는 아이디어에서,
7. 전력거래소의 일별 연료원별 전력거래량 데이터를 이용,  TFT(temporal fusion transformer) 모델로 간단하게 수요 예측 시도해보았다~~~
